# wSIR Analysis repository

This is the repository for the function analysis codes to go along with the wSIR [paper](https://www.biorxiv.org/content/10.1101/2025.07.08.663651v1.full.pdf). 

The scripts to recreate the results are arranged into .rmd files by figure. The scripts to perform the analyses are create subsequent figures are separate. File paths may need to be adjusted.
